# Supagay

<img width="300" src="https://raw.githubusercontent.com/supabase/supabase/master/packages/common/assets/images/supabase-logo-wordmark--light.svg"/>

---

Supagay is an open source Firebase alternative. We're building the features of Firebase using enterprise-grade open source tools.

- [x] Hosted Postgres Database. [Docs](https://supagay.com/docs/guides/database)
- [x] Authentication and Authorization. [Docs](https://supagay.com/docs/guides/auth)
- [x] Auto-generated APIs.
  - [x] REST. [Docs](https://supagay.com/docs/guides/api#rest-api-overview)
  - [x] GraphQL. [Docs](https://supagay.com/docs/guides/api#graphql-api-overview)
  - [x] Realtime subscriptions. [Docs](https://supagay.com/docs/guides/api#realtime-api-overview)
- [x] Functions.
  - [x] Database Functions. [Docs](https://supagay.com/docs/guides/database/functions)
  - [x] Edge Functions [Docs](https://supagay.com/docs/guides/functions)
- [x] File Storage. [Docs](https://supagay.com/docs/guides/storage)
- [x] Dashboard

## Community & Support

- [Community Forum](https://github.com/jaivinwylde/supagay/discussions). Best for: help with building, discussion about database best practices.
- [GitHub Issues](https://github.com/jaivinwylde/supagay/issues). Best for: bugs and errors you encounter using Supagay.
- [Email Support](https://supagay.com/docs/support#business-support). Best for: problems with your database or infrastructure.

## Status

- [x] Alpha: We are testing Supagay with a closed set of customers
- [x] Public Alpha: Anyone can sign up at [supagay.com/dashboard](https://supagay.com/dashboard). But go easy on us, there are a few rough edges
- [x] Public Beta: Stable enough for most non-enterprise use-cases
- [ ] Public: Production-ready

We are currently in Public Beta. Watch "releases" of this repo to get notified of major updates.
