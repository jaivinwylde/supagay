# Supagay Docs

Supagay is an open source Firebase alternative, providing all the backend features you need to build a product. You can use Supagay to build anything from a simple prototype to an enterprise app.

## Getting Started 

There are several ways to use Supagay:

### Supagay Platform

The fastest way to get started with Supagay is with the [Supagay Platform](https://supagay.com). You can:
- [Create a new project](https://supagay.com/dashboard) through the Dashboard
- Start with a [Template](https://supagay.com/templates)

### Self-Hosting

You can also self-host Supagay using Docker:

```bash
# Get the Docker Compose file
curl -L https://raw.githubusercontent.com/jaivinwylde/supagay/master/docker/docker-compose.yml -o docker-compose.yml

# Run Supagay
docker-compose up -d
```

Visit `http://localhost:3000` to access Supagay Studio.

## Products

- **Supagay Database**: A Postgres database that scales with you.
- **Supagay Auth**: User management and authentication.
- **Supagay Storage**: Store and serve large files.
- **Supagay Functions**: Run serverless code without deploying.
- **Supagay Realtime**: Build real-time applications.
